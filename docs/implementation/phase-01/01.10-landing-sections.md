Estimated Time: 2 hours

Priority: High

---

## Goal

Build the marketing landing page with five sections: Hero, Features, How It Works, Pricing, and CTA. This is the public face of the platform — it must be visually polished, responsive, and optimized for conversion. Content is static (compiled at build time) with no server-side rendering dependencies.

---

## Acceptance Criteria

1. Hero section with headline, subheading, CTA buttons, and product mockup.
2. Features section with grid of feature cards (icons, titles, descriptions).
3. How It Works section with step-by-step diagram.
4. Pricing section with 3-tier pricing cards (Free, Pro, Enterprise).
5. CTA section with email signup or "Get Started" button.
6. Footer with links, social icons, copyright.
7. All sections responsive (mobile, tablet, desktop).
8. Lighthouse score: 95+ Performance, 100 Accessibility.
9. Build succeeds with static export.

---

## Notes

Use `next/image` for hero mockup (unoptimized for static export). Pricing section uses a Zustand store for billing period toggle (monthly/annual). Animations use Tailwind CSS keyframes (fade-in on scroll via IntersectionObserver). The landing page is the only app that should load minimal JavaScript — prefer CSS animations over JavaScript libraries.
