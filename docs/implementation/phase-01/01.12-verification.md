# Step

Step ID: 01.12

Step Name: Phase 01 Verification

Estimated Time: 30 minutes

Priority: Medium

Difficulty: Easy

Dependencies: All above (01.01 through 01.11)

---

## Goal

Verify that all four Next.js applications are properly scaffolded, themed, and navigable. Run build, type-check, lint, and visual verification for each app. Update the verification suite from Phase 00 to include Phase 01-specific checks.

---

## Requirements

1. `pnpm build` succeeds for all four apps.
2. `pnpm typecheck` passes for all four apps.
3. `pnpm lint` passes for all four apps.
4. Manual visual verification of each app:
   - Web app: layout renders, sidebar works, theme toggle works, routes navigate
   - Admin app: layout renders, sidebar groups work, routes navigate
   - Docs app: markdown renders, code highlighting works, TOC works
   - Landing app: all sections render, pricing toggle works, responsive
5. Add Phase 01 verification to `scripts/verify/`:
   - Verify each app's `package.json` has correct name (`@repo/web`, `@repo/admin`, etc.)
   - Verify each app's `next.config.ts` exists and is valid
   - Verify each app has a root layout with metadata
   - Verify TailwindCSS theme is consistent across apps
   - Verify no app imports from another app

---

## Acceptance Criteria

1. All builds succeed.
2. All quality checks pass.
3. Visual inspection confirms each app renders correctly.
4. Phase 01 verification tests pass.
