## Goal

Build the thread list sidebar component showing all conversations for the current workspace with search, sorting, and navigation.

## Business Context

The thread list is the primary navigation for the chat interface. Users need to quickly find past conversations by title, see the most recent ones, and create new threads.

## Technical Context

This is a Client Component that uses TanStack Query for data fetching with optimistic updates for thread creation/deletion. The thread list appears in a collapsible left sidebar.

## Requirements

- Fetch paginated thread list for current workspace
- Display thread title, last message preview (truncated), timestamp
- Sort by last activity (most recent first)
- Search/filter by title
- Create new thread button with inline title input
- Delete thread with confirmation dialog
- Active thread highlighting
- Keyboard navigation (up/down arrows, enter to open)
- Responsive: collapsible on mobile, persistent on desktop
- Loading skeleton state
- Empty state: "No conversations yet. Start a new chat."
