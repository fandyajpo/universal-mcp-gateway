## Goal

Build the main message area with streaming message rendering, markdown display, code highlighting, and auto-scroll behavior.

## Business Context

The message area is the core visual component of the chat experience. It displays an ongoing conversation with streaming AI responses, formatted markdown, syntax-highlighted code blocks, and a natural scroll experience.

## Technical Context

This is a Client Component that connects to the SSE streaming endpoint. Messages are stored in TanStack Query's cache and rendered incrementally as tokens arrive.

## Requirements

- Display user and assistant messages in a chat bubble layout
- Streaming message rendering (append tokens as they arrive)
- Markdown rendering with `react-markdown` and `shiki` for code highlighting
- Auto-scroll to bottom for new messages (pause if user has scrolled up)
- "Scroll to bottom" button when scrolled up
- Tool call rendering (show which tool was called, its status, and result)
- Citation rendering (numbered citations linked to source documents)
- Timestamps on messages (relative: "2m ago", absolute on hover)
- Copy code block button
- Loading indicator for AI thinking state
- Error state for failed messages with retry option
