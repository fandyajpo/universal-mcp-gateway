## Goal

Implement chat history features: infinite scroll pagination, search within threads, and conversation export functionality.

## Business Context

As users accumulate conversations, they need to search through past threads, find specific messages, and export conversations for sharing or archival purposes.

## Technical Context

History features leverage the existing repository layer with additional query patterns: full-text search on thread titles and message content, cursor-based pagination for infinite scroll, and export formatting.

## Requirements

- Infinite scroll pagination on thread list (cursor-based)
- Search threads by title (debounced, server-side)
- Search messages within a thread (Ctrl/Cmd+F)
- Export single thread as markdown
- Export single thread as JSON
- Bulk export workspace threads
- Thread archiving (hide from main list but not delete)
- Recently viewed threads (last 5)
- Thread pinning (pin important threads to top)
