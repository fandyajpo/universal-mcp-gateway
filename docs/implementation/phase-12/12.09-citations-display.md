## Goal

Implement citation rendering in chat messages, showing source documents from RAG retrieval with hover previews and click-through links.

## Business Context

RAG-powered responses include citations to source documents. Users need to verify AI claims by examining the original sources. Citations build trust and enable fact-checking.

## Technical Context

Citations are returned as part of the AI response metadata from the RAG engine. Each citation contains a source document ID, title, excerpt, and relevance score. The UI renders them as numbered references with hover previews.

## Requirements

- Numbered inline citations in AI responses: `[1]`, `[2]`
- Citation tooltip on hover showing: document title, excerpt (2-3 lines), relevance score
- Click on citation opens source document in knowledge base viewer
- Citation list at end of message with full details
- Loading state while citation metadata loads
- Fallback for deleted/unauthorized documents: "[Source unavailable]"
- Keyboard accessible (Tab to navigate citations)
- Support for 0 to 20 citations per message
