Estimated Time: 1 hour

---

## Goal

Implement signed URL generation for secure file download. Signed URLs are time-limited (default: 1 hour) and can be optionally scoped to specific IP addresses or user agents. The signing uses AWS Signature V4 via `@aws-sdk/s3-request-presigner`.

---

## Files To Create

- packages/storage/src/signed-urls.ts

Methods: `generateDownloadUrl(key, expiresIn?)` — creates presigned GET URL; `generateUploadUrl(key, contentType, expiresIn?)` — creates presigned PUT URL for direct-to-S3 uploads; `getPublicUrl(key)` — returns public URL (if bucket has public access).

---

## Acceptance Criteria

1. `generateDownloadUrl` returns a URL valid for 1 hour by default.
2. URL is prefixed with R2 public endpoint.
3. URL includes AWS Signature V4 query parameters.
4. After expiry, the URL returns 403 Forbidden.
5. `generateUploadUrl` creates a presigned PUT URL for direct upload (optional, for future use).
6. Signed URLs cannot be used to list or delete objects.
