# Step

Step ID: 05.09

Step Name: Phase 05 Verification

Estimated Time: 30 minutes

Priority: Medium

Difficulty: Easy

Dependencies: All above (05.01 through 05.08)

---

## Goal

Verify the complete storage system: R2 connectivity, file upload/download, type validation, size enforcement, tenant isolation, API routes, and upload UI. Run automated tests and manual smoke tests.

---

## Acceptance Criteria

1. `pnpm test` passes for `@repo/storage` and all storage-related tests.
2. `pnpm build` succeeds for all apps.
3. Manual: upload a PDF file via the upload UI — confirm progress, success state, file appears.
4. Manual: upload a `.exe` renamed to `.pdf` — confirm type validation rejects it.
5. Manual: upload a file larger than workspace limit — confirm 413 error.
6. Manual: click download on an uploaded file — confirm signed URL downloads correctly.
7. Manual: delete a file — confirm it's removed from R2 and MongoDB.
8. Manual: check storage usage stats — confirm they update correctly.
9. Manual: verify tenant isolation — workspace A files not visible in workspace B.
10. Manual: upload from mobile — confirm responsive layout works.
11. Manual: keyboard-only upload — confirm accessibility works.
