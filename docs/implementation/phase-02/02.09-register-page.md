# Step

Step ID: 02.09

Step Name: Register Page

Estimated Time: 1 hour

Priority: High

Difficulty: Medium

Dependencies: 02.03, 01.01

---

## Goal

Build the registration page at `/register` with name, email, password, and confirm password fields. Includes email verification flow (post-registration page) and OAuth sign-up alternative.

---

## Files To Create

- apps/web/src/app/(auth)/register/page.tsx
- apps/web/src/app/(auth)/register/register-form.tsx
- apps/web/src/app/(auth)/verify-email/page.tsx
- apps/web/src/app/(auth)/verify-email/page.tsx (verify token via query param)
- apps/web/src/actions/auth/register.ts

---

## Acceptance Criteria

1. Registration form validates all fields client-side.
2. Password requirements displayed as user types.
3. Confirm password must match password.
4. Successful registration redirects to "Verify your email" page.
5. Email verification link renders "Email verified — redirecting to login" with countdown.
6. OAuth registration flows through same account linking logic.
7. Rate limiting: max 3 registrations per IP per hour.
