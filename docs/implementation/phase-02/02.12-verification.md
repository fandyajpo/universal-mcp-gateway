# Step

Step ID: 02.12

Step Name: Auth Verification

Estimated Time: 30 minutes

Priority: Medium

Difficulty: Easy

Dependencies: All above (02.01 through 02.11)

---

## Goal

Verify the complete authentication system end-to-end: register, verify email, login, OAuth, MFA, password reset, RBAC enforcement, and auth middleware. Run automated integration tests and manual smoke tests.

---

## Acceptance Criteria

1. `pnpm test` passes for `@repo/auth` and auth-related tests.
2. `pnpm build` succeeds for all apps.
3. Manual: register a new user, verify email, login, confirm session works.
4. Manual: login with Google OAuth, confirm account creation.
5. Manual: login with GitHub OAuth, confirm account creation.
6. Manual: enable MFA, logout, login with TOTP code, confirm session.
7. Manual: request password reset, click link, set new password, login.
8. Manual: verify RBAC — viewer user cannot access admin routes.
9. Manual: verify auth middleware redirects unauthenticated requests.
10. Manual: verify rate limiting — 10 rapid login attempts, account locked.
