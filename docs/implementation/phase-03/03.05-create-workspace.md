# Step

Step ID: 03.05

Step Name: Create Workspace Flow

Estimated Time: 1 hour

Priority: High

Difficulty: Medium

Dependencies: 03.04, 01.01

---

## Goal

Build the "Create Workspace" flow: a dialog/modal with a form for workspace name, slug (auto-generated from name, editable), optional description, and avatar upload. On success, redirect to the new workspace.

---

## Files To Create

- apps/web/src/app/_components/workspace/create-workspace-dialog.tsx
- apps/web/src/app/_components/workspace/create-workspace-form.tsx
- apps/web/src/app/_components/workspace/avatar-upload.tsx
- apps/web/src/hooks/use-create-workspace.ts
- apps/web/src/actions/workspace/create.ts

---

## Acceptance Criteria

1. Dialog opens from workspace switcher or onboarding.
2. Form auto-generates slug from name (editable).
3. Slug uniqueness checked in real-time (debounced API call).
4. Avatar upload shows preview before submitting.
5. Form validates on client with Zod.
6. On success, redirects to the new workspace.
7. On error, shows error message without clearing form.
