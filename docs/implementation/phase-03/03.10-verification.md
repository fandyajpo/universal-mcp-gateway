# Step

Step ID: 03.10

Step Name: Phase 03 Verification

Estimated Time: 30 minutes

Priority: Medium

Difficulty: Easy

Dependencies: All above (03.01 through 03.09)

---

## Goal

Verify the complete workspace system: CRUD, member management, invitations, settings, and workspace switcher. Run automated tests and manual smoke tests.

---

## Acceptance Criteria

1. `pnpm test` passes for all workspace-related tests.
2. `pnpm build` succeeds for all apps.
3. Manual: create a workspace, verify slug auto-generation.
4. Manual: invite a user by email, accept invitation, verify membership.
5. Manual: change member role, verify permission changes take effect.
6. Manual: remove a member, verify they can no longer access workspace.
7. Manual: transfer ownership, verify new owner has admin powers.
8. Manual: archive workspace, verify it's hidden from workspace list.
9. Manual: restore archived workspace.
10. Manual: switch between workspaces, verify data isolation.
11. Manual: update workspace settings (name, slug, avatar).
