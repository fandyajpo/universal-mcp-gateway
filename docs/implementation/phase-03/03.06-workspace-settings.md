# Step

Step ID: 03.06

Step Name: Workspace Settings Page

Estimated Time: 1 hour

Priority: Medium

Difficulty: Medium

Dependencies: 03.04, 01.01

---

## Goal

Build the workspace settings page at `/settings/workspace` with sections for general settings (name, slug, avatar, description), workspace management (archive/delete), and workspace-level feature flags.

---

## Files To Create

- apps/web/src/app/settings/workspace/page.tsx
- apps/web/src/app/settings/workspace/general-settings.tsx
- apps/web/src/app/settings/workspace/danger-zone.tsx
- apps/web/src/app/settings/workspace/feature-flags.tsx
- apps/web/src/actions/workspace/update.ts
- apps/web/src/actions/workspace/archive.ts

---

## Acceptance Criteria

1. General settings form is pre-populated with current workspace data.
2. Slug change checks uniqueness.
3. Archive workspace shows confirmation dialog explaining consequences.
4. Archived workspace shows "Restore" option instead of "Archive".
5. Feature flags section shows toggleable flags (if workspace-level flags exist).
6. Only workspace admins see the settings page link.
7. Changes saved show success toast.
