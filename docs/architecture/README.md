# Architecture Documentation

This directory contains detailed architecture documentation for the Universal MCP Gateway platform.

## Contents

### C4 Model

The architecture is documented using the C4 model approach:

1. **Context Diagram** — system boundaries, users, and external systems
2. **Container Diagram** — application containers and their responsibilities
3. **Component Diagram** — internal components of each container
4. **Code** — package-level design

### Key Documents

- `../ARCHITECTURE.md` — Main architecture document with diagrams
- `../adr/` — Architectural Decision Records

### Diagrams

All architecture diagrams use Mermaid syntax and are embedded in the markdown documents.

### Package Architecture

See `../ARCHITECTURE.md` for the complete package dependency graph and detailed architecture description.
