# API Documentation

This directory will contain API reference documentation for the Universal MCP Gateway platform.

## Structure

- `rest/` — REST API endpoint documentation
- `websocket/` — WebSocket/SSE streaming API documentation
- `mcp/` — MCP protocol specification
- `sdk/` — SDK API reference

## Conventions

- OpenAPI 3.1 specification for REST endpoints
- TypeScript type definitions as API contracts
- All APIs are versioned (v1, v2)
- Breaking changes require ADR and migration guide
