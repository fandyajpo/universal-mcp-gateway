# Specifications

This directory contains detailed technical specifications for platform features and components.

## Structure

- `features/` — Feature specifications (RFC format)
- `protocols/` — Protocol specifications (MCP, WebSocket)
- `schemas/` — Data schema specifications
- `integrations/` — Third-party integration specifications

## RFC Process

Major features follow the RFC process:

1. **Draft** — Initial proposal in `features/rfcs/`
2. **Review** — Open for discussion and feedback
3. **Accepted** — Approved for implementation
4. **Implemented** — Feature completed
5. **Deprecated** — No longer relevant

## Templates

Each RFC includes:
- Summary
- Motivation
- Design
- Trade-offs
- Migration plan (if applicable)
