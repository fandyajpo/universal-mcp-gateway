Welcome to the Universal MCP Gateway documentation. This guide will help you get started with the platform.

## Quick Start

Install the CLI tool and initialize your workspace:

```bash
npm install -g @universal-mcp-gateway/cli
umgw init my-workspace
cd my-workspace
```

## Configuration

Create a configuration file:

```yaml
version: "1.0"
workspace:
  name: My Workspace
  slug: my-workspace
auth:
  providers:
    - email
    - github
    - google
```

## API Usage

Here's how to make your first API call:

```ts
import { createClient } from "@universal-mcp-gateway/client";

const client = createClient({
  apiKey: process.env.UMGW_API_KEY,
  workspace: "my-workspace",
});

const response = await client.chat.completions.create({
  model: "gpt-4",
  messages: [
    { role: "user", content: "Hello, world!" },
  ],
});

console.log(response.choices[0]?.message.content);
```

## TypeScript Types

The platform provides full TypeScript support:

```ts
import type {
  Workspace,
  User,
  ApiKey,
  ChatCompletion,
} from "@universal-mcp-gateway/types";

async function getWorkspace(id: string): Promise<Workspace> {
  const response = await fetch(`/api/workspaces/${id}`);
  return response.json() as Promise<Workspace>;
}
```
